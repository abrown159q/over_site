Visual Scroll

Program uses webcamera and hand gesture to scroll a window

Goal: To allow a user to use a hand gesture in order to cause the screen to move on the given window so you can look at recipes without touching your keyboard

Gratitudes and credits

I started this project based off of the work of ZIXUAN WANG found at http://dmprojects.lmc.gatech.edu/~zwang620/ . In it, he builds a gesture control system in order for a user to explore recipes without touching the keyboard. He, however, uses a leap sensor which I did not have. I instead wanted to use only my camera

This project also owes a debt of gratitude to Murtaza Hassan at https://www.murtazahassan.com/ and his 3 hour OpenCV tutorial https://www.murtazahassan.com/learn-opencv-3hours/ for teaching the basics of opencv in python.

Also, the haar file for palm and fist image recognition was taken from: https://github.com/OAID/CVGesture

Also, the pyautogui library was used to control the scrolling in windows. (thanks to Al Sweigart at https://automatetheboringstuff.com/)
